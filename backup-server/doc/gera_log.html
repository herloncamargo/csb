Módulo gera_log.py¶

Módulo responsável pela gravação do registro das atividades realizadas no servidor.

Autor: Herlon Camargo

Data: 16/10/2012

modules.gera_log.gera_log(texto)[source]¶

Função que registra no servidor as atividades realizadas referentes ao backup. Recebe um “texto” que, junto com a data e o horário, serão escritos no arquivo “backup-server.log”, localizado no diretório corrente onde o script “backup_server.py” foi executado.
